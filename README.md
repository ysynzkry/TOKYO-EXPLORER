# 🗼 TOKYO EXPLORER

An immersive 3D web experience showcasing Tokyo's culture, landmarks, and beauty.

![Tokyo Explorer](./preview.png)

## ✨ Features

- **3D Landmarks** - Interactive rotating 3D models of Tokyo's famous sites
- **Akira-inspired Design** - Bold typography with black, pink, and red gradients
- **Smooth Animations** - GSAP and Framer Motion powered transitions
- **Interactive Experiences** - Personalized trip recommendations
- **Virtual Walk** - Google Street View integration
- **Trivia Game** - Test your Tokyo knowledge
- **Tokyo × Cairo** - Cultural connections between two ancient cities

## 🛠️ Tech Stack

- **Frontend**: React, Three.js, Framer Motion, GSAP, Tailwind CSS
- **Backend**: Node.js, Express
- **3D**: @react-three/fiber, @react-three/drei

## 🚀 Quick Start

```bash
# Clone the repository
git clone [https://github.com/ysynzkry/tokyo-explorer.git](https://github.com/ysynzkry/tokyo-explorer.git)

# Install frontend dependencies
cd tokyo-explorer/client
npm install

# Install backend dependencies
cd ../server
npm install

# Run development servers
# Terminal 1 - Frontend
cd client && npm start

# Terminal 2 - Backend
cd server && npm run dev
